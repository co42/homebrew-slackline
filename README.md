# homebrew-slackline

Homebrew tap for [slackline](https://github.com/co42/slackline) - a read-only Slack CLI for AI agents.

## Install

```bash
brew install co42/slackline/slackline
```

Or tap first:

```bash
brew tap co42/slackline
brew install slackline
```

## Update

```bash
brew upgrade slackline
```
